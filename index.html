Инструкция по использованию меню

  1. Меню открывается по клику на пункте с треугольником
  2. Подменю может содержать вложенные подменю (до 3 уровней)
  3. При клике на пункте меню без подменю происходит переход по ссылке
  4. Для закрытия меню кликните повторно на пункт или на другой элемент меню
  5. Меню адаптировано для мобильных устройств
